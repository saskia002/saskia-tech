## Getting Started

Dvelopment server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

#### Generating NEXTAUTH_SECRET

Open git bash on windows or linux bash:
`openssl rand -base64 32`
